About

Hi there! I'm Rachael, and I am a full-stack web developer with a background in customer service, most recently in the health and fitness industry as a personal trainer. Some previous limited experience I had with HTML sparked an interest in web development that I was thrilled to pursue, and the more I learn the more I feel at home in the industry.

Having the knowledge and skills to make websites and applications come to life from the ground up gratifies a part of me that truly enjoys problem-solving and the production process. I am an effective communicator, collaborator, critical thinker, and project organizer, and I hope these skills along with my continued education in the field will bolster my capability as a web dev and help me help others bring their ideas to life on the web efficiently and creatively.

Pokédex App

Overview

This project focused mainly on using JavaScript to load Pokémon data from an external API, as well as implementing the UI framework Bootstrap to create a clean-looking, user-friendly interface. Users can view a list of Pokémon, click on a button to view more details about a specific Pokémon on a pop-up modal, as well as utilize a search function.

  • [HTML]
  • [CSS]

Meet App

Overview

For this project I was tasked with building a serverless, progressive web app with React using a test-driven development technique. It uses the Google Calendar API to fetch upcoming events and filters events by city. Users are able to show/hide event details, specify the number of events shown, as well as use the app while offline.

Technologies:
  • [React]
